# Documentation

## Installation

You can install Flyapi in one of the following ways:

<Tabs>
  <Tab label="NPM">
		```bash
		npm add @steelwindshellahillz/flyapi
		```
	</Tab>
  <Tab label="PNPM">
		```bash
		pnpm add @steelwindshellahillz/flyapi
		```
	</Tab>
  <Tab label="YARN">
		```bash
		yarn add @steelwindshellahillz/flyapi
		```
	</Tab>
</Tabs>
